本项目使用 [Create React App](https://github.com/facebookincubator/create-react-app)构建。
## 说明 ##
本项目使用API来自斗鱼官方论坛。
CSS部分：使用`styled-components`，css in js方案；
Javascript框架：`React``;
http请求：`axios`，可选es6新特性Promise替代；
路由：  `React-router`。
支持响应式。
## 克隆代码到本地 ##
```
git clone git@github.com:fishhtml/FCC.git
```
## 进入目录 ##
运行：
```
npm run start
```
